使用 srcset 还是 media-query？

srcset，media-query 配合 display: none 不会禁止浏览器下载未展示的图片，需要再搭配 html 的懒加载属性 loading="lazy"，这个属性有兼容问题。media-query 是 CSS 特性，如果用户的浏览器禁止了 CSS，将无法应用图片自适应，而 srcset 没有这些问题。
