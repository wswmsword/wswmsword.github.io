hanav

npm install hanav

hanav 是一个 React 导航栏组件库，包含一组触发器和一组对应的菜单面板，用户可以通过触发器展开、切换、收起菜单面板。导航栏通常出现在网站的顶部，提供最希望用户访问的链接和其它控件。

🍯

流畅的过渡动画

符合直觉的入场、退场和切换动画，退场动画结束后，消失的面板将被移除 DOM

🎹

键盘导航

按下 Tab 和 Esc，让焦点流动，用户可以任意选择鼠标或键盘访问

♿️

屏幕阅读器导航

有规范的 ARIA 属性，具备可访问性，可以通过 TalkBack 或 VoiceOver 辅助设备访问

🎨

高度自定义

被渲染的 DOM 和组件一一对应，为每个元素设置样式，还能自定义切换过渡动画

📱

兼容移动端设计

提供了专为移动端设计的一系列组件，选中“迷你模式”或切换移动端设备体验效果

🚀

开发体验良好

使用形态和渲染结果的结构一致，不同的功能对应不同的组件，简单、易懂、高效

hanav 的使用方法很简单，提供了直观简单的 API，有良好的开发体验，具体的使用方法请查看 hanav 的 GitHub 仓库主页。
