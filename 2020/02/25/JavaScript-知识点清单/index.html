闭包和作用域

产生闭包的两种表现

  • 函数作为参数被传递
  • 函数作为返回值被返回
作为参数
function create() {
  let a = 100;
  return function() {
    console.log(a);
  }
}
let fn = create();
let a = 200;
fn(); // 100
作为返回值
function print(fn) {
  let a = 200;
  fn();
}
let a = 100;
function fn() {
  console.log(a);
}
print(fn) // 100

形成闭包，执行闭包里的函数，那么所使用的变量将从函数定义的域向上查找，而不是在执行函数所在的域。把形成的闭包想象成一颗气泡，会把域包裹进来并隔离闭包外的变量。

this

  • 作为普通函数
  • 使用 call apply bind
  • 作为对象被调用
  • 在 class 方法中调用
  • 箭头函数

和闭包相反，this 取的值由函数执行的域确认，而不是函数定义的域。

this 有时会导致奇怪的结果：当调用全局方法时注意，全局方法内部的函数 this 指向 window；为了响应 ES6 来解决 this 问题可能使用箭头函数，但是箭头函数作为函数无法通过 apply, call, bind 等方法指定其 this；

箭头函数里的 this：箭头函数没有自己的 this，会继承上层作用域的 this

模拟 bind
Function.prototype.bind1 = function() {
    const args = Array.prototype.slice.call(arguments);
  const t = args.shift();
  const self = this;
  return function() {
    return self.apply(t, args);
  }
}

  • apply 以数组或集合的方式传入

  const args, res;
  t = t || window;

实践

  • 目标：多个实例（这里是存在 conveyo 数组里的 DOM 元素）的横向滚动动画。
  • 细节：使用 requestAnimationFrame 或 setInterval。
    • 实现对多个实例的不同动画效果，如果直接调用函数则所有函数将共享同一个域的变量，导致每一个实例的行为相同，因此使用闭包。requestAnimationFrame 执行过程的关键在于它的动画逻辑函数里仍需调用 rAF，因此动画逻辑函数不能是匿名函数，在同属闭包域内声明这个动画逻辑函数，rAF 下一次执行它的时候便也可以继续使用闭包里的变量。
    • 和 rAF不同，setInterval 只需传入 1 次函数参数，然后 sI 反复执行这个函数，因此直接返回逻辑函数，形成闭包。

深拷贝

为了避免浅拷贝和直接引用地址使用深拷贝。

function deepClone(onj = {}) {
  if(typeof obj !== 'object' || obj === null) {
    return obj;
  }

  let result;
  if (obj instanceof Array) {
    result = [];
  } else {
    result = {};
  }

  for (let key in obj) {
    if (obj.hasOwnProperty(key)) {
      result[key] = deepClone(obj[key]);
    }
  }
  return result;
}

递归深入，跳出递归的条件：参数不是对象类型（除了 null 对象）。

数组拍平

function flat(arr) {
  var isDeep = arr.some(function(item) {
    return item instanceof Array;
  }); // 判断数组有否内嵌
  if (!isDeep) {
    return arr;
  }
  var res = Array.prototype.concat.apply([], arr);
  return flat(res);
}

如果数组没有内嵌数组，使用 concat 可以把所有一层的数组直接串联起来，这里的Array.prototype.concat.apply([], arr)把 arr 和 [] 串联。在数组内嵌的情况下进行递归即可平摊数组。

节流

function throttle(fn, delay = 500, context) {
  var timer = null;
  return function() {
    if (timer) {
      return ;
    }
    timer = setTimeout(function() {
      fn.apply(context, arguments);
      timer = null;
    })
  }
}

防抖

function debounce(fn, delay = 500, context) {
  var timer = null;
  return function() {
    if (timer) {
      clearTimeout(timer);
    }
    var self = context;
    timer = setTimeout(function() {
      fn.apply(self, arguments);
      timer = null;
    }, delay)
  }
}
