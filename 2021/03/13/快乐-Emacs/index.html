（草稿）

在 Emacs 中和计算机沟通就像透过画家的眼睛欣赏世界，并且形形色色（奇奇怪怪）的人正使用着 Emacs，他们专注其中的样子让你不禁斜起 45 度看看究竟发生了什么。这些作为开始 Emacs 的理由确实有点冲动和单薄了，那就再补充一点：尾随直觉吧（“Intuition takes me there”, Intuition, Mind Games, John Lennon），相遇最重要（貌似更单薄了）。

话虽如此，Emacs 却实在有打动人（至少我）的地方：一触即发的上古编辑器之战；化身魔法师挥动魔棒执行函数；继承使用者的战斗力；臆想中的末世编辑器……当然，还有那张 Emacs 嚣张的螺旋线学习曲线图。不过，更有人爱谈 Emacs“作为操作系统”和“煮咖啡”。

总之，Emacs 是一个多才多艺的编辑器。

Emacs 有内置的帮助文档。M-x，~ help-with-tutorial-spec-language，Chinese-GB。

🪄Emacs 魔法

在我想查看“数的转换”相关内容的时候，可以输入：C-h i m elisp RET m Numberic Conversions RET，以后以这个模式描述：info elisp - Numberic Conversions。

在我想查看“位运算”的相关内容的时候：info elisp - Bitwise Operations on Integers。

Emacs 解释 举例 来源

C-x b 切换 buffer
C-x k 杀死 buffer

Windows

Emacs 解释 举例 来源
C-x 2 垂直切割当前 Window 上下布局
C-x 3 水平切割当前 Window 两侧布局
C-x o 切换至其他 Window 配合C-x 2，把焦点转移到新窗口
C-x 0 关闭当前 Window
C-x 1 关闭当前以外的所有 Window
